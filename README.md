# 创业Idea Agent Swarm

一个基于 Claude Code 的多Agent创业Idea生成与分析系统。提供参考材料（文章、痛点描述、行业趋势），系统将通过**两轮迭代**自动生成6-10个创业idea并调度9个专业Agent对每个idea进行全方位分析。

## 核心特点：两轮迭代式生成

与一次性生成不同，本系统采用**两轮迭代策略**：
1. **第一轮**：基于机会扫描生成3-5个idea → 完整8维度分析
2. **经验提炼**：总结R1分析中的高分方向、意外风险、团队洞察
3. **第二轮**：idea-generator吸收R1全量分析经验 → 生成3-5个全新idea → 完整分析
4. **统一排名**：两轮共6-10个idea按综合评分统一排序

第二轮idea受益于第一轮的分析洞察，质量更高、方向更精准。

## 快速开始

### 前置条件
- 已安装 [Claude Code](https://docs.anthropic.com/en/docs/claude-code)
- Claude Code 版本 ≥ 1.0

### 使用方式

在此文件夹中打开终端，运行：

```bash
claude
```

然后直接提供你的参考材料，例如：

```
我最近读到一篇文章，提到AI基础设施还停留在上一个时代，
很多企业想用AI但发现现有的数据管道、模型部署工具都不好用。
请基于这个趋势帮我发现创业机会。
```

系统会自动启动 `startup-swarm` Agent，通过两轮迭代生成6-10个创业idea，并调度9个专业Agent对每个idea进行深度分析。

### 也可以指定使用特定Agent

```bash
# 调用完整流程（两轮迭代生成idea + 分析）
claude "请用 startup-swarm 基于这篇文章分析创业机会：[你的参考材料]"

# 只调用市场扫描
claude "请用 market-analyst 扫描模式分析这个领域的市场机会：[你的描述]"

# 只调用技术架构分析
claude "请用 tech-architect 评估这个方案的技术可行性：[你的描述]"
```

## Agent 团队成员

| Agent | 角色 | 核心能力 | 模式 |
|-------|------|---------|------|
| **startup-swarm** | 一键入口 | 自动调度全流程（两轮迭代生成+分析） | - |
| **orchestrator** | 主协调员 | 分解任务、调度Agent、汇总结论 | - |
| **idea-generator** | 创意策略师 | 综合信号生成创业idea、排序推荐 | 首轮/迭代 |
| **market-analyst** | 市场分析师 | TAM/SAM/SOM、竞品分析、趋势研判 | 扫描/分析 |
| **tech-architect** | 技术架构师 | 技术可行性、MVP方案、技术壁垒 | - |
| **business-model** | 商业模式专家 | 商业画布、盈利模式、单位经济模型 | - |
| **ux-researcher** | 用户体验研究员 | 用户画像、需求验证、用户旅程 | 扫描/分析 |
| **investor-advisor** | 投资人视角 | 融资策略、估值分析、退出路径 | - |
| **legal-compliance** | 法务合规顾问 | 法律风险、数据合规、知识产权 | - |
| **growth-hacker** | 增长黑客 | 获客策略、增长飞轮、病毒传播 | - |
| **domain-expert** | 行业领域专家 | 产业链分析、行业趋势、跨行业启发 | 扫描/分析 |

    └─────┬─────┘  └──────┬──────┘  └──────┬──────┘

                ┌─────────▼─────────┐

                          │

              ║  第一轮分析 (R1)     ║
              ════════════╪════════════
                          │
          ┌───────────────┼────────────────┐

          │      × R1每个idea并行          │
    ┌─────▼─────┐  ┌──────▼──────┐  ┌──────▼──────┐

    │行业专家    │  │ 市场分析师   │  │用户研究员    │

    │ 投资人视角  │  │法务合规顾问  │
    └─────┬─────┘  └──────┬──────┘
          │               │
          └───────┬───────┘
                  │
           ┌──────▼──────┐
           │ 轮次4: R1   │
           │ 评分汇总     │
           │ + 经验提炼   │
           └──────┬──────┘
                  │
              ════╪════
              ║ 迭代 ║
              ════╪════
                  │
        ┌─────────▼─────────┐
        │ 轮次5: 第二轮Idea  │
        │  创意策略师[迭代]   │
        │  吸收R1经验        │

        └─────────┬─────────┘
                  │
      ════════════╪════════════

      ════════════╪════════════

          ┌───────┼───────┐
          │ 轮次6: R2深度  │

## 输出文件

分析完成后，报告保存在 `output/` 目录：

- `output/detailed_report.md` - 详细分析报告
  - 第一部分：参考材料与机会扫描
  - 第二部分：全部创业Idea总览与排名（⚠️两轮共6-10个idea统一排序，标注R1/R2来源）
  - 第三部分：逐个Idea深度分析（按排名顺序，每个idea完整8维度分析）
  - 第四部分：两轮迭代洞察（R2相比R1的进步、迭代中的关键发现）
  - 第五部分：综合对比与最终建议
- `output/summary_report.md` - 简要总结报告（2000字以内）
  - 先展示全部Idea排名和详细介绍（标注R1/R2来源）
  - 再逐个展示分析概要、迭代洞察和最终推荐

## 评分维度

| 维度 | 权重 | 说明 |
|------|------|------|
| **创新性** | **10%** | **idea的新颖程度和差异化** |
| 市场机会 | 18% | 市场规模、增速、竞争格局 |
| 用户需求 | 14% | 需求真实性、频次、强度 |
| 技术可行性 | 13% | 实现难度、技术壁垒 |
| 商业模式 | 13% | 盈利能力、单位经济 |
| 增长潜力 | 10% | 获客效率、增长飞轮 |
| 投资吸引力 | 9% | 融资可行性、退出路径 |
| 行业机会 | 8% | 产业链位置、行业趋势 |
| 法律合规 | 5% | 法律风险、合规成本 |

## 自定义

你可以根据需要修改 `.claude/agents/` 下的Agent配置文件，调整分析框架、增减Agent角色、或修改评分权重。
